2. JavaScript Unit 2

Since April 18, I started learning JavaScript Unit 2, which includes three lessons: 2.1 - Functions, 2.2 - Return Statement and 2.3 - Scopes. In these three assignments, I mastered building functions and distinguished between functions and return functions. In the scopes lesson, I mastered global scopes and local scopes, where global scopes are defined outside the function and can be called anywhere in the program, while local scopes are defined in the function and can only be called in the function.
